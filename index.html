ShellApiLogOptimizer

ShellApiLogOptimizer 是一款专为 ShellApi 设计的日志优化插件，旨在显著提升日志查询和统计的速度。

在 GitHub 上查看项目

主要特性

高效查询

优化日志查询算法，大幅提升查询速度，让您更快地获取所需信息。

快速统计

改进统计方法，使复杂的日志统计操作变得更加迅速和高效。

易于集成

设计为插件形式，可以轻松集成到现有的 ShellApi 系统中，无需大规模修改。

系统要求

  • 建议配置：4核CPU，8GB内存以上（32GB以上最佳）
  • 推荐使用 Docker Compose 进行部署

快速开始

  1. 克隆项目仓库：
    git clone https://github.com/akl7777777/ShellApiLogOptimizer.git
  2. 进入 docker-compose/environment 目录
  3. 执行 docker compose up -d 启动环境
  4. 检查 Docker 进程状态，确保启动成功
  5. 返回项目根目录，执行 docker compose up -d 启动项目

项目将在 8080 端口运行。如遇到权限问题，请尝试执行 chmod 777 -R data。
